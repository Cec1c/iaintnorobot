# I Ain't No Robot

一个 AstrBot 单群自然插话插件原型。

它的目标不是让机器人更能聊，而是让它**少说、短说、像普通群友一样偶尔接一句**。插件只使用 AstrBot 已有的消息、配置、LLM 和发送能力，不起外部服务，也不单独接外部 LLM。

## 特性

- 单群观察：当前原型只服务一个群，适合先压低复杂度测试效果。
- WebUI 选群：配置面板维护“已添加群聊”，插件页面只从已添加群聊中选择目标群。
- 轻量记忆：使用插件数据目录里的 SQLite，只保存短文本上下文和压缩摘要。
- 低频插话：后台周期检查，本地规则先判断是否值得开口。
- 话题优先：触发时先看有没有可附和或可轻聊的话题，没有再低概率自创一句。
- 人味短句：限制一句话、限制字数，过滤 Markdown、客服腔和常见 AI 套话。
- 黑话学习：尝试记忆群内黑话解释，不确定时避开使用。
- 内线询问：可在 WebUI 指定内线 QQ，用来询问不确定黑话或依赖图片/游戏机制的内容。
- 身份处理：不会承认自己是 AI；被骂 AI 时可以不理、轻嘲或短怼。
- 插件甩锅：被问到账号插件触发时，倾向一句话带过，例如“我只是拿这个号说话”。

## 指令

```text
/iar status   查看状态
/iar on       开启当前群
/iar off      关闭当前群
/iar addgroup 把当前群加入 WebUI 可选群聊
/iar groups   查看已添加群聊
/iar select   选择已添加群聊作为目标
/iar say      强制尝试生成一句，方便测试
/iar summary  手动刷新群语境记忆
/iar slang    查看黑话记忆
/iar learn    手动扫描黑话
/iar reset    清空当前群的插件记忆
```

## 安装简述

把本目录放到 AstrBot 的 `data/plugins/astrbot_plugin_i_aint_no_robot` 下，然后在 WebUI 重载插件。
