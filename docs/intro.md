# 项目介绍

I Ain't No Robot 是一个 AstrBot 插件原型，用来让机器人在一个群里低频、短句、自然地插话。

设计重点是克制：

- 不把每条消息都交给 LLM。
- 不把群聊完整历史当知识库存起来。
- 不输出长段解释、列表、总结或客服式回答。
- 不承认自己是 AI、机器人、模型或插件。
- 被说像 AI、被骂 AI、被问插件行为时，可以不理，也可以像普通群友一样短怼或糊弄过去。

## 工作流

```text
群消息进入
  -> 插件记录短期纯文本
  -> 后台定时醒来
  -> 本地规则判断是否活跃、是否太频繁
  -> 必要时调用 AstrBot 当前 LLM
  -> 清洗输出
  -> 合格才发一句
```

## 记忆策略

插件只做轻量记忆：

- 最近消息：默认每群最多 200 条，且只存纯文本。
- 群话题摘要：例如最近常聊什么。
- 群语气摘要：例如随意、嘴碎、爱吐槽、游戏群等。
- 机器人发言记录：避免短时间重复说同一句。

SQLite 文件位于 AstrBot 插件数据目录中。它不是外部服务，不需要用户额外部署数据库。

## 输出风格

输出目标是“像真人懒得打字”：

- 一句话。
- 字数短。
- 不自证。
- 不装懂。
- 能闭嘴就闭嘴。
- 可以半截话、轻吐槽、短附和。

当遇到账号插件行为相关问题时，插件会倾向于生成短句解释，例如：

```text
我就拿这号说两句
插件那套别问我
那触发跟我没啥关系
```
